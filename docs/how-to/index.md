# How-to guides

How-to guides are goal-oriented recipes.
Each one answers a specific "How do I…?" question.
They assume you have a working snpArcher installation and basic command-line familiarity.

| Guide | Goal |
|---|---|
| [Install snpArcher](install.md) | Get snpArcher, Snakemake, and conda/mamba set up |
| [Create a sample sheet](sample-sheet.md) | Build the CSV that tells snpArcher about your samples |
| [Configure your run](configure.md) | Set up `config.yaml` for your dataset |
| [Run locally](run-local.md) | Execute on a workstation |
| [Run on HPC](run-hpc.md) | Execute on SLURM or other schedulers |
| [Interpret QC reports](qc-report.md) | Read the interactive QC dashboard |
| [Filter and postprocess](postprocess.md) | Filter your VCF by samples, sites, and callable regions |
| [Add a custom module](custom-modules.md) | Extend snpArcher with your own Snakemake module |
| [Troubleshoot errors](troubleshoot.md) | Fix common problems |
